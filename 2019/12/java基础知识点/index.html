Java基础知识点

十二月 14, 2019 发布在 JavaSE

Java语言特点

1.跨平台
2.安全性
3.面向对象
4.分布式
5.健壮性

JDK和JRE

1.JDK是指开发者环境，包括了JRE,Java的基本类库和开发环境。

2.JRE是指运行环境，只有java的基本类库。

3.编译Java文件使用JDK的javac命令，编译后的Java文件会产生一个.class文件，再通过JRE的java命令便可运行。

标识符命名规则

1.标识符指Java中所有可以起名字的，比如文件名、类名、变量名…

2.标识符由字母、下划线(_)、数字和$组成。

3.注意事项：不能以数字开头；不能是关键字；可以包含有中文和￥，但建议不使用，不符合规范。

数据类型

1.基本数据类型

  • 整型

    名称 关键字 字节数 范围值
    字节型 byte 1 (8bit) -128~127
    短整型 short 2 -2^15 ~ 2^15-1
    整型 int 4 -2^31 ~ 2^31-1
    长整型 long 8 -2^63 ~ 2^63-1

  • 浮点型

    名称 关键字 字节数
    单精度 float 4
    双精度 double 8

  • 字符型

    关键字为char，占2个字节，只能为一个字符，并且该字符用单引号引起来。

  • 布尔型

    关键字为boolean，占1个字节，范围只有true/false，只能产生正负。

2.引用数据类型

  • 类(class)：Object，String，Date…

  • 数组(Array)

  • 接口(interface)：List，Map…

定义变量

  • 语法1：数据类型 变量名;

      变量名=值;

    int num;
    num=1;
  • 语法2：数据类型 变量名=值;
    int num=1;
  • 语法3：数据类型 变量名1=值1，变量名2=值2，变量名3;
    int first=1，second=2，third;

注意事项：局部变量必须赋值，否则程序会报错；全局变量有默认值。

数据类型转换

  • 精度排序(从小到大):byte,short,char,int,float,double.

    注意事项：小转大，自动转；大转小，要强转，即赋值时前面加上括号写上要强转的类型

    short+int =int;//小转大
int a=2;
double b=(double)a;//大转小

运算符

1.算术运算符：+ - * / % ++ –

2.关系运算符：== != > < >= <=

3.逻辑运算符：&& || !

&&和&、||和|的区别：&&和&都是表示与，区别是&&若第一个条件不满足，后面条件就不再判断。而&要对所有的条件都进行判断，||和|同理。

4.赋值运算符：= += -= /= %= «=(左移赋值运算符) »= &=(按位与赋值符) ^= |=

5.条件运算符(三元运算符)：boolean条件？为true时返回：为false时返回。
